## Getting Started

### Prerequisites
- Node.js 18+
- Firebase project with Firestore

### Installation

```bash
git clone https://github.com/mayalok-ventures/deeplink-creators.git
cd deeplink-creators
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
```
